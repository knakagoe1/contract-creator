# contract-creator
## set up
1. npm install
2. bower install
3. npm run serve -- --inline
4. http://localhost:8080
## execute
1. input "Solidity Code"
2. input "Node URL" like http://localhost:8545
3. input "Contract Name"
4. click "create" button
## info
- contract owner : eth.accounts[0]
- gas : 4700000
